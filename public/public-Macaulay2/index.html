Welcome to Macaulay2

This web site allows you to try, learn, and use Macaulay2, without the need to install it. Get started by selecting a tutorial below. To learn how to use this site effectively, try the Welcome tutorial. Have fun!

Macaulay2 is an open source software system devoted to supporting research in algebraic geometry and commutative algebra, and related field in mathematics or applications.
